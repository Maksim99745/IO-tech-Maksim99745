# Финальная проверка Permissions

## ✅ Что вижу на скриншоте:

Все permissions настроены правильно:
- ✅ Client: READ
- ✅ Page: READ
- ✅ Service: READ
- ✅ Subscriber: CREATE + READ
- ✅ Team-member: READ
- ✅ User: READ

---

## ⚠️ Возможные проблемы:

### 1. Глобальные настройки CREATE

Вижу, что в глобальных настройках CREATE частично отмечен (синий квадрат с черточкой). Это может вызывать проблемы.

**Решение:**
- Снимите галочку с глобального CREATE (сделайте его пустым)
- Оставьте только индивидуальные галочки для каждой коллекции
- Нажмите **Save**

### 2. Нужно сохранить

**ВАЖНО:** Нажмите кнопку **Save** в правом верхнем углу и дождитесь подтверждения!

### 3. Перезапустить Strapi

После сохранения **обязательно перезапустите Strapi**:

```bash
cd strapi-backend
# Остановите (Ctrl+C)
npm run develop
```

---

## Проверка после исправления:

1. **В браузере откройте:**
   - http://localhost:1337/api/services

2. **Должен вернуться JSON:**
   ```json
   {"data":[],"meta":{"pagination":{...}}}
   ```

3. **Если все еще 401:**
   - Проверьте, что нажали Save
   - Перезапустите Strapi
   - Попробуйте снять и поставить галочки заново

---

## Альтернативная проверка:

Попробуйте через curl:
```bash
curl http://localhost:1337/api/services
```

Если видите JSON - все работает!  
Если видите 401 - нужно перезапустить Strapi или проверить глобальные настройки.
